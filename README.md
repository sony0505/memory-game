114-2 USR 共創長者幸福成果競賽

參賽組別：學習成果展示組

作品名稱

數位任意門：憶起打造腦力防護罩！

一、 創作動機

在快速變遷的數位時代，台灣的長輩常面臨著兩大挑戰：第一是「數位代溝」，現代生活充斥著 Wi-Fi、QR Code、行動支付等新科技，許多長者因不熟悉而感到焦慮，甚至覺得與年輕人的世界脫節；第二是「認知退化」，長者若缺乏適當的腦力刺激，容易面臨記憶力衰退，增加失智症（老年癡呆）的風險。
我們看見了阿公阿嬤想學習卻又害怕受挫的心情。因此，我們運用資工專長，設計了一款「免安裝、零門檻、充滿鼓勵」的網頁互動配對遊戲。希望長輩在充滿趣味的遊戲過程中，不僅能鍛鍊記憶力，更能輕鬆認識現代科技名詞，縮短青銀世代間的距離。

二、 解決問題與痛點

本作品精準對應長者在現代生活中的痛點，提出三大解決方案：

消弭科技恐懼（無痛上手）： 長輩常因害怕按錯或操作複雜而排斥數位工具。本系統採用 Web 網頁技術，不需下載任何 APP，只需掃描 QR Code 即可一秒開啟，極大地降低了長輩接觸新科技的門檻與心理負擔。

減緩認知老化（活化大腦）： 採用經典且易懂的「記憶翻牌配對」機制。長輩需要在遊戲中記住不同圖卡的位置並進行配對。這種重複提取記憶的過程，能有效刺激大腦海馬迴，活化神經突觸，達到預防失智的「腦力防護」效果。

認識現代知識（跨越代溝）： 將日常生活中必備的現代知識（如：Wi-Fi 無線網路、行動支付、視訊通話等）設計成遊戲圖卡。讓長輩在配對成功的成就感中，自然而然地記住這些名詞與功用，未來聽到年輕人討論時不再感到陌生，創造更多青銀共樂的話題。

三、 系統特色與核心功能

我們以「高齡友善 UX/UI」為核心原則，開發出此套互動系統：

視覺無障礙設計： 採用高對比色彩、超大字體與直觀的真實圖片，徹底解決長輩老花眼、視力模糊導致的閱讀困難。

Web Speech API 即時語音回饋： 系統內建語音合成技術。當長輩翻開卡片或配對成功時，瀏覽器會自動以親切的中文語音發出「答對了！這是視訊通話，可以從螢幕上看到遠方的孫子喔！」等鼓勵與教學，給予聽覺刺激與正向的情感支持。

跨平台雲端部署： 系統具備響應式設計（RWD），能完美適配各式手機與大螢幕平板，並透過 GitHub Pages 進行穩定託管，隨開即玩。

四、 預期效益與社會影響

個體效益（健康與自信）： 透過持續的腦力訓練，強化長輩的認知儲備（Cognitive Reserve）。更重要的是，在遊戲中獲得的成就感能提升長者的「數位自信心」，證明他們依然有能力學習新事物。

社會效益（青銀共創）： 此作品是非常優良的「跨世代破冰工具」。在推廣時，可由年輕學子（如大學生或孫輩）擔任「科技小老師」引導長輩遊玩，藉由遊戲互動增加陪伴的時間與品質，落實 USR 大學社會責任，共創長者幸福人生。

五、 未來展望

目前系統已具備高度完整性與現場展示能力。未來，我們計畫擴充「客製化題庫模組」，允許社區日照中心或家屬根據長者的興趣，自行上傳生活照片或不同的學習圖卡。同時，也期盼導入輕量級數據紀錄，觀察長者完成遊戲的反應時間變化，作為日常認知健康評估的參考指標，持續以科技溫暖高齡社會。
